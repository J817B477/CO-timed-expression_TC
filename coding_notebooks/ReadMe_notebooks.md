the coding notebooks directory organizes jupyter notebooks for secondary differential expression analysis and (eventually) wgcna
- it includes prescripted ipynb files
- will include pdf renderings of notebooks as analysis is finalized
- additional analytic tools like visuals, and csvs will be stored in either the de_analysis or WGCNAnalysis folders based on respective notebooks
- all of the analytics tools will have context in the analysis of each notebook but will have separate pdf renderings as well in the appropriate analysis directory
- different html versions were rendered based on whether or not code was included; all can be opened in the browser
- best steps for generating pdf version is still being determined: support for latex math notation is currently a challenge
- notebooks have their own text for explanations, evaluations and justifications for the coding steps taken within it; the environment allows for coding and reporting on the results in the same file.
