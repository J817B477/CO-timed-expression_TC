### scripts/ directory
- contains all scripts used to support rules/jobs of [snakemake](../snakefile) pipeline that had script directives in any of the rules
1. [scripts/generate_overrep_fa.py](../scripts/generate_overrep_fa.py)
	- supports 'get_overrep_fa' rule
	- checks all sequences determined to be overrepresented, but with unknown source ("No Hit" in FastQC Reports) using NCBI BLAST to determine if the sequence is human
	- writes all overrepresented sequences determined to to be non-biological signal to a fastA file for use in trimming
1. [scripts/infer_strand.py](../scripts/infer_strand.py)
	- determines what the consensus is for strand direction in files generated by determining which direction is most commonly determined for each sample's bam file using infer_experiment.py (from RSeQC)
    - uses the strings of individual strand/ folder member files to to accumulate the number of times each file is determined to be forward or reverse
    - only returns zero is all percentages of strands of either direction sum to less than 66% in all samples
    - other wise which direction is most frequently observed
1. [scripts/clean_counts.R](../scripts/clean_counts.R):
    - supports clean_counts snakemake rule
    - creates nTPMs and updates column names in data extracted from counts.txt so that that DESeq metadata ('coldata') can be parsed from them
    - outputs: 
        - "updated_sampleNames_map.csv" 
            - contains parallel columns with pairings indicating what new name replaced the old name so that it can be demonstrated the identified of samples were not swapped
        - "de_analysis/counts_matrix.csv" 
            - for DESeq2 analysis
        - "de_analysis/tpm_counts.csv" 
            - for presenting counts in normalized convention and deseq2 analysis

1. [scripts/DEanalysis_expandedContrasts.R](../scripts/DEanalysis_expandedContrasts.R):
    - generates test results for experimental group contrasts and exports them for secondary analysis
    - the cross product of all experimental group contrasts was created while ensuring that the base of each contrast was logically consistent with the extent of treatment exposure for each experimental group
    - results were outputed as csv:
        - [expandedContrastsPADJ_tpm_counts.csv](../de_analysis/expandedContrastsPADJ_tpm_counts.csv) includes adjusted p-value variables foe each contrast along with tpm counts
        - [expandedContrastsDIRECTION_tpm_counts.csv](../de_analysis/expandedContrastsDIRECTION_tpm_counts.csv)
            - values are strings : "upregulated", "downregulated" or "not significant" providing information about direction or perturbation if there was a significant expression level
        - [expandedContrastsL2FC_tpm_counts.csv](../de_analysis/expandedContrastsL2FC_tpm_counts.csv)
            - values are $log_{2}(fc)$ where fc is the fold change (the factor by which the expression is increased or decreased)
            - this includes the all the information in the csv but also includes magnitude (the previous csv is convenient for some downstream coding tasks) 
        - [expandedContrasts_AllTestData_tpm_counts.csv](../de_analysis/expandedContrasts_AllTestData_tpm_counts.csv)
            - provides all of the data for each contrast that was generated by DESeq statistical testing
    - visuals
        - per test sample dist matrix heatmap and sample pca plot
            - checks group replicate proximity and separation
            - validates replicates represent the same experimental group well
        - volcano plots for each contrast
